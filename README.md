 # Small x86 Assembly interpreter to Brainfk
   I wrote a small brainfuck interpreter in x86 Assembly using resb, not resq. It supports core instructions such as +, -, >, <, [, ], and ., I'm not using , for now. I used buffers for bracket matching and outputs. It detects unmatched brackets, underflow and overflow of buffers. Used 20, 000 cells for now. It set some conditions lower than 20,000. Please read the notes on my code (small x86 Assembly interpreter to BFK) to understand it. 

   NOTE: This is PURE Assembly, no integration of other programming languages such as .py
